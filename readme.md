
🚀 Battle Strategy Optimizer
Greedy vs Dynamic Programming Simulator

📌 Overview
Battle Strategy Optimizer is an interactive simulation that compares Greedy and Dynamic Programming (DP) approaches in resource-constrained decision-making scenarios.
It helps visualize how different strategies impact outcomes when allocating limited soldiers against multiple enemies.

🎯 Objective
Maximize total reward while efficiently managing limited soldiers and understanding the difference between fast decisions and optimal solutions.

🧠 Algorithms Used
🔹 Greedy Algorithm
Selects best immediate option (reward/cost)
Fast but not always optimal
🔹 Dynamic Programming (0/1 Knapsack)
Evaluates all combinations
Guarantees optimal result

⚙️ How It Works
Enter total soldiers
Add enemies with cost & reward
System runs:
Greedy strategy
DP optimization
Displays performance comparison
